# 定向运动轨迹日记功能

## 🎯 功能概述

将定向运动轨迹绘制系统升级为个人日记形式，让每次定向运动都成为珍贵的回忆记录。

## ✨ 新增功能

### 1. **日记式界面**

- 将原来的点位列表改为时间轴式日记界面
- 每个点位都可以添加个人备注和感受
- 分段描述功能，记录每段路线的意义

### 2. **特殊标记系统**

- **右键点击画布**添加特殊标记
- 支持 5 种标记类型：
  - 📝 **备注** - 记录重要信息
  - 📷 **照片** - 标记拍照位置
  - ⚠️ **警告** - 标记危险区域
  - ✓ **检查点** - 标记检查点位置
  - ⭐ **里程碑** - 标记重要节点

### 3. **PDF 导出功能**

- 一键导出完整的轨迹日记
- 包含：
  - 轨迹图
  - 点位信息
  - 特殊标记
  - 个人描述
  - 时间记录

## 🎮 使用方法

### 绘制轨迹

1. 点击"开始绘制"按钮
2. 在画布上点击添加点位
3. 自动记录点位坐标和类型（直线/曲线）

### 添加标记

1. 在画布上**右键点击**
2. 选择标记类型
3. 输入标记内容
4. 标记会显示在画布和日记列表中

### 编辑日记

1. 在右侧日记面板中编辑分段描述
2. 为每个点位添加个人备注
3. 查看和管理特殊标记

### 导出 PDF

1. 点击日记面板右上角的"导出 PDF"按钮
2. 系统自动生成包含所有信息的 PDF 文件
3. 文件自动下载到本地

## 🔧 技术实现

### 数据结构

```typescript
interface Segment {
  id: string
  name: string
  points: Point[]
  markers: Marker[]
  description: string
  createdAt: Date
  updatedAt: Date
}

interface Point {
  x: number
  y: number
  type?: 'line' | 'curve'
  description?: string
  timestamp?: Date
}

interface Marker {
  id: string
  x: number
  y: number
  type: 'note' | 'photo' | 'warning' | 'checkpoint' | 'milestone'
  content: string
  timestamp: Date
}
```

### 核心组件

- `DiaryList.vue` - 日记式列表组件
- `Canvas.vue` - 增强的画布组件（支持标记）
- `PDFExporter` - PDF 导出工具类
